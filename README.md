Chef AI 🍳

An AI-powered recipe generator that transforms the ingredients you already have into complete recipes using React and Hugging Face AI. Simply add the ingredients available in your kitchen, and Chef AI generates a detailed recipe with ingredients, instructions, and serving suggestions.

📖 Overview

Chef AI helps users reduce food waste and discover new meal ideas by generating recipes based on available ingredients. The application leverages Hugging Face's AI models to create personalized cooking instructions in real time.

Whether you have a few basic ingredients or a fully stocked kitchen, Chef AI can suggest delicious recipes without requiring extensive meal planning.

✨ Features
🥗 Add and manage available ingredients
🤖 AI-powered recipe generation using Hugging Face
📋 Detailed ingredient lists
👨‍🍳 Step-by-step cooking instructions
🍽️ Serving recommendations
⚡ Fast and responsive React-based interface
🎨 Clean and intuitive user experience

🚀 Installation
1. Clone the Repository
git clone https://github.com/yourusername/chef-ai.git
cd chef-ai
2. Install Dependencies
npm install
3. Configure Environment Variables

Create a .env file in the root directory:

VITE_HF_API_KEY=your_huggingface_api_key

or

REACT_APP_HF_API_KEY=your_huggingface_api_key

depending on your project setup.

4. Start Development Server
npm run dev

or

npm start

🔄 How It Works
User enters ingredients available in their kitchen.
Ingredients are stored in React state.
When Get a Recipe is clicked:
The ingredient list is sent to the Hugging Face AI model.
The model generates a recipe based on the provided ingredients.
The generated recipe is displayed dynamically on the page.
